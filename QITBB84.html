So what is BB84 Protocol??

To Start...

BB84 protocol is a form of quantum key distribution (QKD)

Whats quantum key distribution?

Secure communication method that uses cyptographic protocol involving quantum mechanic components (qubits)

Whats cryptography?

Defining this in the classical sense, it is a method of storing & transmitting data in a certain form so that the only ones whom can read and proces it are the ones it is intended for

Two major branches

Secret/Symmetric

Two parties encrypt/decrpt the message with the SAME shared key in secret-key cryptography.

The key must be shared physically, by email, so on so somebody might have stole the key while sharring it.

SIGNIFICANT DISADVANTAGE

Public/Asymmetric

The key has a public AND private part.

The data encrypted with the public key can be decrypted with the private key

Data encypted with private key can only be solved by public key
